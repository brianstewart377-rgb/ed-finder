# Transfer ED:Finder to Raspberry Pi

## Method 1: SCP from Windows (Recommended)

```cmd
scp "%USERPROFILE%\Downloads\ed-finder-FINAL-COMPLETE.tar.gz" mightyraith@192.168.0.115:~/
```

**Alternative paths if the file is elsewhere:**
```cmd
scp "%USERPROFILE%\Desktop\ed-finder-FINAL-COMPLETE.tar.gz" mightyraith@192.168.0.115:~/
scp "C:\Users\YourUsername\Downloads\ed-finder-FINAL-COMPLETE.tar.gz" mightyraith@192.168.0.115:~/
```

## Method 2: Direct Download on Pi

```bash
# From anywhere on your Pi
cd ~
wget [DOWNLOAD_URL] -O ed-finder-FINAL-COMPLETE.tar.gz
```

## After Transfer

```bash
# Extract
cd ~
tar -xzf ed-finder-FINAL-COMPLETE.tar.gz
cd ed-finder-FINAL-COMPLETE

# Deploy
./ULTIMATE_FIX.sh
```

## Verify Transfer

```bash
# Check file exists
ls -lh ~/ed-finder-FINAL-COMPLETE.tar.gz

# Check file size (should be ~130 KB)
du -h ~/ed-finder-FINAL-COMPLETE.tar.gz
```

## Troubleshooting SCP

### If you get "connection refused":
```bash
# On the Pi, ensure SSH is running:
sudo systemctl status ssh
sudo systemctl start ssh
```

### If you get "permission denied":
- Check username is correct: `mightyraith`
- Check IP is correct: `192.168.0.115`
- Try: `ssh mightyraith@192.168.0.115` to test connection

### If file ends up in wrong location:
```bash
# Find it
find ~ -name "ed-finder-FINAL-COMPLETE.tar.gz" 2>/dev/null

# Move it to home
mv /path/to/ed-finder-FINAL-COMPLETE.tar.gz ~/
```
